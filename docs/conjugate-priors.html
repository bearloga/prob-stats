22 Conjugate Priors

Hello there.

Figure 22.1: Progress of streaming updates with conjugate priors. There is an initial prior \(\mbox{beta}(\alpha_0, \beta_0)\) and a stream of data \(y_1, y_2, \ldots, y_N.\) After each data point \(y_n\) is observed, the prior parameters \(\alpha_{n-1}, \beta_{n-1}\) are updated to the posterior parameters \(\alpha_{n}, \beta_n,\) which then acts as a prior for subsequent data.
